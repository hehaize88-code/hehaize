How to review this category

Separate print choice, garment size and current price.

01Exact print and color option02Chest width and body length03Collar, seam and print QC

T-shirt pages can look simple while hiding dozens of selectable designs. Use each local reference page to identify the recorded catalog item and source ID, then reopen the live destination to confirm which design is attached to the displayed price. Save the selected print, color and size together. Otherwise a later warehouse image may be impossible to compare with the option that was actually submitted.

Measure a T-shirt you already own from armpit to armpit and from shoulder to hem, then compare those centimetre figures with the seller chart. During QC, check overall proportions, collar shape, seam direction, print centering and visible stains. A ruler request should name the exact measurement needed. T-shirts normally use parcel space efficiently, but light-colored items should be separated from dirty packaging or dark materials that could transfer marks.

Warehouse QC focus

Front and back photographs should be straight enough to judge overall proportions. A ruler across the chest or length answers more than a generic request for better photos.
